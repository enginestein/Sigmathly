# Sigmathly

**The first ever advanced mathematical framework for assembly**

# Installation

• Go to releases tab

• Download .dll release / .so release / .a release

• Link the release to assembly file 

``` bash  gcc main.c -o program -L/path/to/release -lsigmathly ```

OR

``` bash gcc main.c -o program.exe -L/path/to/library -lmylibrary ```
