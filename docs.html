Sigmath Framework Documentation

Welcome to the documentation for the Sigmath framework! This page contains information on how to use the framework and its various components. Use the links below to navigate to different sections of the documentation.

Installation

There are few ways to install Sigmath:

Get releases from Github

.DLL release:-

For visual studio projects add .DLL reference in your project.

For python: $ pip install pythonnet 


            import clr 
            clr.AddReference('path/to/Sigmath.dll')

            from Sigmath import Basic
            print(Basic.Add(2, 2))

NuGet release:-

$ nuget install Sigmath.1.0.0.nupkg -Source "path-to-folder-containing-nupkg" 
$ py -m pip install Sigmath
$ nuget install Sigmath

Wheel release:-

$ pip install Sigmath-1.0.0-py3-none-any.whl

TAR GZ release:-

$ pip install Sigmath-1.0.0.tar.gz

Other ways

$ pip install Sigmath
$ nuget install Sigmath
$ choco install Sigmath
$ brew install Sigmath

This function can be used to add an infinite amount of numbers. All arguements must be double data type.

Sets class

This class contains operations on Sets. In python, the arguement must be a list.

Sigmath.Sets.Union<T>(Sets1, Sets2)

This function returns a new list that contains all elements from both input lists with duplicates removed.


            • C# Console.WriteLine(Sigmath.Sets.Union(Sets1, Sets2))
            • Python print(Sigmath.Sets.union(Sets1, Sets2))

Sigmath.Sets.Intersection<T>(Sets1, Sets2)

This function returns a new list that contains only the elements that are common to both input lists.


            • C# Console.WriteLine(Sigmath.Sets.Intersection(Sets1, Sets2))
            • Python print(Sigmath.Sets.intersection(Sets1, Sets2))

Sigmath.Sets.Difference<T>(Sets1, Sets2)

This function returns a new list that contains only the elements that are in the first input list but not in the second.


            • C# Console.WriteLine(Sigmath.Sets.Difference(Sets1, Sets2))
            • Python print(Sigmath.Sets.diff(Sets1, Sets2))

Sigmath.Sets.SymmetricDifference<T>(Sets1, Sets2)

This function returns a new list that contains only the elements that are in one of the input lists, but not in both.


            • C# Console.WriteLine(Sigmath.Sets.SymmetricDifference(Sets1, Sets2))
            • Python print(Sigmath.Sets.symdiff(Sets1, Sets2))

Sigmath.Sets.IsSubSets<T>(subSets, Sets)

This function returns true if all elements in the first input list are also present in the second input list.


            • C# Console.WriteLine(Sigmath.Sets.IsSubSets(subSets, Sets))
            • Python print(Sigmath.Sets.subset(subSets, Sets))

Sigmath.Sets.IsSuperSets<T>(superSets, Sets)

This function returns true if all elements in the second input list are also present in the first input list.


            • C# Console.WriteLine(Sigmath.Sets.IsSuperSets(superSets, Sets))
            • Python print(Sigmath.Sets.superset(superSets, Sets))

Calculus class

This class containts various calculus equation solving methods.

Calculus.Derivative(Func<double, double> f, double x)

This function computes the derivative of a given function at a specified point using a numerical method.

Parameters

  • f - a function to differentiate
  • x - the point at which to find the derivative

Returns

The numerical derivative of f at x.


            • C# Console.WriteLine(Calculus.Derivative(f, x))
            • C++ std::cout << Calculus::Derivative(f, x) << std::endl;
            • Python print(Calculus.derivative(f, x))

Calculus.Integral(Func<double, double> f, double a, double b, int n)

This function computes the definite integral of a given function over a specified interval using the trapezoidal rule.

Parameters

  • f - a function to integrate
  • a - the lower limit of integration
  • b - the upper limit of integration
  • n - the number of subintervals to use in the trapezoidal rule

Returns

The numerical value of the definite integral of f over the interval [a, b].


            • C# Console.WriteLine(Calculus.Integral(f, a, b, n))
            • C++ std::cout << Calculus::Integral(f, a, b, n) << std::endl;
            • Python print(Calculus.integral(f, a, b, n))

Calculus.Limit(Func<double, double> f, Func<double, double> g, double x)

This function computes the limit of a given function as it approaches a specified point using L'Hopital's Rule.

Parameters

  • f - the numerator function
  • g - the denominator function
  • x - the point at which to find the limit

Returns

The numerical value of the limit of f / g as x approaches the specified point.


            • C# Console.WriteLine(Calculus.Limit(f, g, x))
            • C++ std::cout << Calculus::Limit(f, g, x) << std::endl;
            • Python print(Calculus.limit(f, g, x))

Calculus.FindRoot(f, a, b, epsilon)

This function uses the bisection method to find a root of the input function within a specified interval, with a specified tolerance level.

  • f: a function taking a single input argument and returning a double
  • a: the left endpoint of the interval to search for a root in
  • b: the right endpoint of the interval to search for a root in
  • epsilon (optional): the tolerance level for the root, defaults to 1e-6

            • C# Console.WriteLine(Calculus.FindRoot(f, a, b, epsilon))
            • C++ cout << Calculus::FindRoot(f, a, b, epsilon) << endl;
            • Python print(calculus.find_root(f, a, b, epsilon))

Calculus.PartialDerivative(f, index, h)

This function returns a function representing the partial derivative of the input function with respect to the specified variable, using the central difference method with a specified step size.

  • f: a function taking an array of input arguments and returning a double
  • index: the index of the input argument to differentiate with respect to
  • h (optional): the step size to use for the finite difference method, defaults to 1e-6

            • C# Func<double[], double> dfdx = Calculus.PartialDerivative(f, index, h);
              double result = dfdx(x);
            • C++ auto dfdx = Calculus::PartialDerivative(f, index, h);
              double result = dfdx(x);
            • Python dfdx = calculus.partial_derivative(f, index, h)
              result = dfdx(x)

Calculus.Gradient(f, h)

This function returns a function representing the gradient of the input function, using the central difference method with a specified step size.

  • f: a function taking an array of input arguments and returning a double
  • h (optional): the step size to use for the finite difference method, defaults to 1e-6

            • C# Func<double[], double[]> gradf = Calculus.Gradient(f, h);
              double[] result = gradf(x);
            • C++ auto gradf = Calculus::Gradient(f, h);
              vector<double> result = gradf(x);
            • Python gradf = calculus.gradient(f, h)
              result = gradf(x)

Calculus.Hessian(f, h = 1e-6)

This function returns the Hessian matrix of a multivariable function, given as a lambda expression. The Hessian matrix contains all second-order partial derivatives of the function.

Parameters:

  • f: The function of which to calculate the Hessian.
  • h (optional): The step size used for numerical differentiation. Default value is 1e-6.

Returns:

  • A lambda expression that calculates the Hessian matrix at a given point.

            • C# var hessian = Calculus.Hessian(f);
              double[,] hessianMatrix = hessian(x);
            • Python hessian = Calculus.hessian(f)
              hessianMatrix = hessian(x)

Calculus.FindCriticalPoints(f, a, b, epsilon = 1e-6)

This function finds all critical points of a single-variable function on a given interval, using the bisection method. A critical point is a point at which the derivative of the function is zero or undefined.

Parameters:

  • f: The function of which to find the critical points.
  • a: The left endpoint of the interval.
  • b: The right endpoint of the interval.
  • epsilon (optional): The tolerance for the bisection method. Default value is 1e-6.

Returns:

  • An array containing all critical points of the function on the interval.

            • C# double[] criticalPoints = Calculus.FindCriticalPoints(f, a, b);
              foreach (double x in criticalPoints) Console.WriteLine(x);
            • Python criticalPoints = Calculus.find_critical_points(f, a, b)
              for x in criticalPoints: print(x)

Calculus.RiemannSum(f, a, b, n, type = RiemannSumType.Left)

This function computes the Riemann sum of a function f over the interval [a, b] using n subintervals and the specified type of Riemann sum. The default type is left Riemann sum.

The available types of Riemann sum are:

  • RiemannSumType.Left - left Riemann sum
  • RiemannSumType.Right - right Riemann sum
  • RiemannSumType.Midpoint - midpoint Riemann sum
  • RiemannSumType.Trapezoidal - trapezoidal Riemann sum

            • C# double result = Calculus.RiemannSum(f, a, b, n, type);
            • C++ double result = Calculus::RiemannSum(f, a, b, n, type);
            • Python result = Calculus.RiemannSum(f, a, b, n, type)

Calculus.DefiniteIntegral(f, a, b, n = 1000)

This function computes the definite integral of a function f over the interval [a, b] using the midpoint Riemann sum with n subintervals.


            • C# double result = Calculus.DefiniteIntegral(f, a, b, n);
            • C++ double result = Calculus::DefiniteIntegral(f, a, b, n);
            • Python result = Calculus.DefiniteIntegral(f, a, b, n)

This function computes the inverse of a square matrix using Gaussian elimination.

Parameters

  • Matrix matrix: the matrix to be inverted

Returns

  • A new Matrix object representing the inverse of the input matrix.

Example Usage
